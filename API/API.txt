Welcome to the Decider API! Here you will find everything that you can use to make decider work.

Section 0: Error Codes
----------------------------------------------------------------------------------------
### Error Codes
    101 - Invalid Email
    102 - Invalid Password
    103 - Invalid Parameters
    104 - Invalid Cookie

Section 1: Models
----------------------------------------------------------------------------------------
### User Model
{
    "fbid"  : Facebook id token
    "email" : user activated email 
    "name"  : user display name
}

### Partial Event Model
{
    "eid"   : unique event id
    "admin_fbid" : administrative user fbid for this event
    "title"      : the title of the event
}

### Event Model
{
    "eid"    : unique event id
    "admin_fbid"  : administrative user fbid for this event
    "title"       : the title of the event
    "description" : the description of the event
}

### Partial Todo Model
{
    "eid"         : the parent event of the todo
    "tdid"        : unique todo id
    "title"       : the title of the todo
    "location"    : partial location model associated with the todo
}


### ToDo Model
{
    "eid"         : the parent event of the todo
    "tdid"        : unique todo id
    "description" : the description of the todo
    "title"       : the title of the todo
    "location"    : location model associated with the todo
}

### Partial Location Model
{
    "lid"           : unique location id
    "title"         : the title of the location
    "lat"           : the latitude of the location
    "lon"           : the longitude of the location
}

### Location Model
{
    "lid"           : unique location id
    "title"         : the title of the location
    "street"        : the street of the location
    "street_number" : the number of the street
    "city"          : the city the location is in
    "state"         : the state the location is in
    "lat"           : the latitude of the location - optional, null if not found
    "lon"           : the longitude of the location - optional, null if not found
}



Section 2: Queries
----------------------------------------------------------------------------------------
### User Authentication [GET] https://decider.azurewindows.com/api/login.php
# Required
    basic HTTP authentication - "basic email:password"
# Optional
    time : the time the session token is to be valid in minutes

# Return
{
    "error_code" : the error code of the request
    "result"     : if successful a user model, null otherwise
}


### User Registration [POST] https://decider.azurewindows.com/api/register.php
# Required
    email    : the user email that is to be registered
    password : the user password that is set to be put into the database
    name     : the display name of the newly created user
# Optional
    time : the time the session token is to be valid in minutes

# Return
{
    "error_code" : the error code of the request
    "result"     : if successful a user model, null otherwise
}

### Create Event [POST] https://decider.azurewindows.com/api/event.php

# Return
